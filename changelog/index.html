chrome extension (v0.2.2) available on github or chrome webstore

v0.2.4 (2014.07.16)

Previous algorithm for sun altitude calculations did not take timezones properly into account, this has now been fixed. Additionally, the algorithm has been improved and should now work as desired anytime and everywhere on the globe.

v0.2.3 (2014.07.16)

Added an on/off toggle to the interface to allow disabling the overlay while storing the prefered opacity.

Limited opacity range to [0, 0.8], since anything above 0.8 is unusable.

Added 20ms linear fade-in of changes applied to the overlay, for a better feel when modifying the settings.
